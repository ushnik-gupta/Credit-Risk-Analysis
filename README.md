Credit Risk Analysis

Tools & Technologies: Python, SQL, MySQL, Excel, Pandas, NumPy, Scikit-Learn, Logistic Regression, Matplotlib, Seaborn, Git, GitHub

Built a Credit Risk Analytics and Loan Default Prediction platform for banking and financial risk assessment. Designed SQL-based analytics to evaluate credit scores, loan approvals, customer income profiles, and risk categories. Developed an Excel dashboard for loan portfolio monitoring and risk reporting. Performed exploratory data analysis and correlation analysis on borrower characteristics. Implemented a Logistic Regression model to predict loan approval/default risk using customer financial attributes. Automated data extraction, reporting, and Excel exports using Python.

Model Performance:

Logistic Regression Classification Model Evaluation Metrics: Accuracy: ~85–90% Precision: ~84% Recall: ~86% F1-Score: ~85% Model validated using train-test split methodology.

Business Insights:

Credit Score and Income emerged as the strongest predictors of loan approval. High-risk applicants exhibited lower credit scores and shorter employment histories. The model supported data-driven loan approval decisions and risk mitigation strategies.
